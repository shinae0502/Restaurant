# Restaurant

## 소개
안녕하세요 안드로이드 개발 스터디 프로젝트 입니다.

스터디 날짜 및 시간 : 매주 토일요일 오후1시 ~ 오후 4시

비용 : 6천원 (매주 수요일 오후 2시까지 입금 기준으로 예약)

준비물 : 노트북
